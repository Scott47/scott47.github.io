Our Story Maps

Front-end Capstone--Using React with ArcGIS for developers/20 Aug. 2019

Our Story Maps is a single page application using React, JSON-server, and ArcGIS API for developers. I made it because I love maps, I wanted to work with this API, and I thought it would be cool for people to be able to tell their stories about place by combining text, interactive maps ,and other multimedia context. The app demonstrates full CRUD functionality. User story elements, including basemaps, locations, and images are stored using JSON-server.

XC

Final Capstone: Full stack Django/React app for Cross Country Coaches / 8 Nov. 2019

XC Stats is a full stack Django/React app that allows high school cross country coaches to manage runner, team, and event information. Coaches can build and edit rosters, track runner contact information, meet times, runner pace, personal records on a specific course, etc. User data, teams, runners, meets, and meet stats are stored using a SQLite database management system.
